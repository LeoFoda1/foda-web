1
El problema

¿Para qué sirve Flexbox?

Cuando ponés elementos en HTML, se apilan uno abajo del otro.
Flexbox te permite ponerlos en fila, centrarlos, distribuirlos... con solo unas líneas de CSS.

SIN flexbox

📦 Caja 1
📦 Caja 2
📦 Caja 3

Se apilan hacia abajo 😩

CON flexbox

📦 1
📦 2
📦 3

¡En fila automáticamente! 🎉

Solo necesitás agregar display: flex al contenedor (la caja que envuelve a los elementos). ¡Eso ya los pone en fila!

/* Esto va en tu CSS */
.contenedor {
  display: flex;
}
💡 Clave: Flexbox se aplica al padre, no a los hijos. El padre es el que manda dónde van sus hijos.

6
flex-wrap

¿Qué pasa si no entran?

Por defecto, todos los elementos se meten en una sola fila aunque no entren. Con flex-wrap: wrap dejan que bajen a la siguiente línea.

flex-wrap: nowrap

Resumen

Todo en un lugar

Las propiedades que aprendiste hoy:

🧩 Propiedades de Flexbox
display: flex
Activa el flexbox en el contenedor. El punto de partida.
flex-direction
row column — Define si los elementos van en fila o columna.
justify-content
center space-between — Distribuye los elementos en el eje principal.
align-items
center flex-start — Alinea los elementos en el eje secundario (altura).
gap
Espacio entre elementos. Ejemplo: gap: 16px
flex-wrap
wrap — Permite que los elementos bajen a la línea siguiente si no entran.
🚀

Desafío final

Copiá este código en tu index.html y modificá las propiedades para que quede centrado y con espacio entre las tarjetas.

/* En tu CSS */
.tarjetas {
  display: flex;
  justify-content: ___; /* completá */
  align-items: ___; /* completá */
  gap: ___;
}

⭐ Extra: Agregá flex-wrap: wrap y hacé que sea una grilla de tarjetas.
